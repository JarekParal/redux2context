# Reactizer

The ultimate boilerplate for [React](https://facebook.github.io/react/)! :dolls:

### Overview

* Flow
* Styled components
* Webpack + hot reloading
* Isomorphic & static rendering
* ESlint

### Scripts

- `yarn start` - Development
- `yarn bundle && yarn run` - Production

Static analysis:
- `yarn test`
- `yarn lint`
- `yarn flow`
